# 语言的构造级别
- 词 word
- 短语 phrase
- 句子 sentence
- 段落 paragraph
- 篇章 discourse


# 名词短语定义
- 名词与它的修饰语一起即构成名词短语


# 名词短语构造
- 左二右六
  * 前置定语或定语
  * 后置定语
  * 公式
     * 限定语+形容词+中心名词（headword）+六类右置定语（介词短语、分词短语、不定式短语、形容词短语、定语从句、同位语从句
     * 较少情况有副词做右置定语
     


# 句子五成分论

主语、谓语、宾语、补语、状语

定语用来修饰或限定名词的。属于名词短语的构成成分。称为短语成分
