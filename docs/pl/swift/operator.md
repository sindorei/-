# 运算符

## 算术运算符
- `+`、`-`、`*`、`/`、`%`
- 不同数据类型不能进行运算
- `isMultiple(of:)`方法可用来判断是否能整除
```swift
var number = 123
let isMultiple = number.isMultiple(of: 7)
```


## 组合复制运算符
- `+=`
- `-=`
- swift 不支持 `++`和`--`


## 比较运算符
- `==`
- `!=`
- `>`
- `<`
- `>=`
- `<=`
- `===` 恒等，用来判断2个对象是否是同一个实例
- `!==`不恒等

## 逻辑运算符
- `!`
- `&&`
- `||`

## 三元运算符
- `? :`

## 区间操作符
- `..<` 半开区间，不包含最后的值
- `...` 闭区间，第一个和最后一个都包含
