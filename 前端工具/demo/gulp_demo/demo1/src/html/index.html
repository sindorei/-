盐城南洋机场位于盐城市亭湖区南洋镇境内，距市中心8.3公里。始建于1958年.是江苏省继南京禄口机场后第二家一类航空口岸。
目前，机场主要航线有首尔、台北、北京、上海、广州、太原、昆明、长沙、西安、武汉、沈阳、哈尔滨、厦门、深圳、天津等。
